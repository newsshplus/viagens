O painel le data/price_history.json, atualizado pelo scripts/run_searches.py a cada execucao do GitHub Actions. Se ainda nao rodou nenhuma busca, ou o arquivo nao carregar, ele cai de volta pros dados de exemplo abaixo. A coluna Avaliacao compara o preco atual de cada rota com a media historica dela no periodo selecionado acima - precisa de pelo menos 2 execucoes registradas pra rota aparecer com avaliacao (antes disso mostra "sem dados"). O formulario grava um novo perfil direto no config/profiles.json do GitHub e ja dispara a busca, via uma funcao serverless (Netlify ou Vercel, detectado automaticamente).
